charitypins

The Idea

The idea behind charitypins started in 2012 when @gsvolt and his friends started The Akron Makerspace (formerly synhak). We questioned.. "are there any other charities in Akron, Ohio that serve the maker community?". Sadly, there were no such resources to be found.

The Requirements

Even though there are websites online as of today that deal with collecting information about charities (like how they manage their funds etc.), there isn't a resource online that can provide a map-first user interface for finding charities near you. Logically, the following requirements were set to direct the project:

  • The website will provide a map interface and will accept one input - source street address
  • The output of the website will be pinned charities that IRS designates in good status near that source street address

Data Source

IRS on their website provides a list of all charities that according to them can accept donations.
